# ft_transcendence — Frontend

React SPA for a social network platform.

**Stack:** Vite · React 18 · TypeScript · Tailwind CSS v4 · shadcn/ui · React Router v7 · TanStack Query v5 · Zustand · i18next · Socket.io-client

## Quick start

```bash
npm install
npm run dev        # http://localhost:5173
```

Requires the backend stack to be running (`docker compose up --build` from the project root).
